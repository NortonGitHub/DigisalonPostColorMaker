# DigisalonPostColorMarker
デジモン創作物共有サイト[「デジモン創作サロン」](https://www.digimonsalon.com/)の利便性の向上を目的とした拡張機能

## Features
トップページに表示される新着記事のうち、最後にアクセスした後に投稿された記事の色を変えて表示します。
